# 🧩 Manim Plugins

Extend the functionality of Manim with these community-driven plugins.

## 🛠 Functional Plugins
- [manim-physics](https://github.com/ManimCommunity/manim-physics) - Rigid bodies, electromagnetism, and waves.
- [manim-chemistry](https://github.com/ManimCommunity/manim-chemistry) - Visualize chemical structures and reactions.
- [manim-slides](https://github.com/jeertmans/manim-slides) - Create interactive presentations using Manim.

## 🎨 Aesthetic & Layout
- [manim-rubikscube](https://github.com/ManimCommunity/manim-rubikscube) - Visualize Rubik's cubes and algorithms.
- [manim-voiceover](https://github.com/ManimCommunity/manim-voiceover) - Easily add AI-generated or recorded voiceovers to your scenes.

## 📊 Data & Plotting
- [manim-data-structures](https://github.com/ManimCommunity/manim-data-structures) - Simplifies the visualization of common data structures.
